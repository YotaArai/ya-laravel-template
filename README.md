# ya-laravel-template

個人開発で用いる個人用リポジトリになります。

## 概要
- Dockerでの開発環境構築(https://qiita.com/YotaArai/items/840934e84e0256ecd06d)
- ヘッダーやフッターなど全体レイアウトはvuetifyで実装
- Twitter,Facebook,Googleでのソーシャルログイン実装済み
- laravel-mixでのcssとjsファイルのgzip圧縮
